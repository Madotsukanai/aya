Aya
===

kilo をベースにして作られた日本語Linux環境向けテキストエディタです。
基本的な使い方はkilo と変わりませんが、一部機能の追加と強化が図られています。

## 導入方法
これを行う前に、[xclip](https://github.com/astrand/xclip)をインストールして下さい。
```
git clone https://github.com/madotsukanai/aya
cd aya
make
```

インストールする場合は、
```
sudo make install
```
